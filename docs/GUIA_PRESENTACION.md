# 🎤 Guía de Presentación - Outfit Catalog

## Objetivo
Este documento organiza qué mostrar el sábado para que el primer adelanto quede claro y ordenado.

---

## Orden sugerido

1. Introducción del proyecto.
2. Mostrar GitHub con las ramas `master`, `staging` y `qa`.
3. Mostrar Jira con las historias de usuario y sus sub-actividades.
4. Mostrar las capturas del app en el documento.
5. Hacer una demo breve de la app si está disponible.

## Qué decir de cada captura

- **Registro / Acceso**: muestra el flujo inicial y la selección de rol.
- **Menú lateral**: evidencia la navegación interna de la app.
- **Galería / Catálogo**: muestra la lista de prendas y filtros.
- **Base local**: demuestra que la persistencia está en SQLite.
- **Panel de usuario**: muestra el dashboard principal.
- **Colecciones / Looks**: demuestra la funcionalidad de looks.
- **Favoritos**: muestra la funcionalidad de marcados favoritos.
